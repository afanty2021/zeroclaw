# Gateway Module — HTTP 网关服务

**模块路径**: `src/gateway/`

## 概述

Gateway 模块提供基于 Axum 的 HTTP 网关服务，支持 Webhook、SSE、WebSocket 和静态文件服务，具有完整的安全防护和超时机制。

## 核心功能

### 1. 安全特性

- **请求体限制**: 最大 64KB
- **请求超时**: 30 秒（防止 slow-loris 攻击）
- **HTTP/1.1 合规性**: 由 hyper/hyper 提供
- **Content-Length 验证**: 自动验证
- **头部清理**: 由 axum/hyper 处理

### 2. 路由端点

- `POST /webhook/:channel`: Webhook 接收
- `GET /ws/:channel`: WebSocket 连接
- `GET /sse/:session`: Server-Sent Events
- `GET /`: 静态文件服务
- `GET /health`: 健康检查

### 3. WebSocket 支持

实时双向通信：
- 按会话隔离
- 消息序列化
- 错误处理

### 4. SSE 流式

Server-Sent Events：
- 实时事件推送
- 自动重连
- 事件类型过滤

### 5. 配对机制

安全配对流程：
- 常量时间比较（防止时序攻击）
- 公共绑定检测
- 配对令牌验证

### 6. API 端点

RESTful API：
- `GET /api/sessions`: 会话列表
- `POST /api/sessions`: 创建会话
- `DELETE /api/sessions/:id`: 删除会话
- `GET /api/sessions/:id/messages`: 消息历史
