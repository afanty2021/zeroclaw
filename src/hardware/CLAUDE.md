# Hardware Module — 硬件发现与枚举

**模块路径**: `src/hardware/`

## 概述

Hardware 模块提供 USB 设备枚举和硬件发现功能，支持自动检测开发板和设备。

## 核心功能

### 1. DiscoveredDevice

发现的设备信息：
```rust
pub struct DiscoveredDevice {
    pub name: String,           // 设备名称（如 "nucleo-f401re"）
    pub detail: Option<String>, // 产品描述
    pub device_path: Option<String>, // 设备路径
    pub transport: HardwareTransport,
}
```

### 2. HardwareTransport

传输类型：
```rust
pub enum HardwareTransport {
    Native,  // 原生架构
    Serial,  // 串口连接
    Probe,   // 调试探针（如 ST-Link）
}
```

### 3. 自动发现

```rust
pub fn discover_hardware() -> Vec<DiscoveredDevice>
```

返回连接的 USB 设备列表。

### 4. CLI 命令

- `hardware discover`: 列出 USB 设备
- `hardware introspect <path>`: 检查设备详情
- `hardware info --chip <chip>`: 获取芯片信息（需要 probe 功能）

### 5. 向导集成

- 推荐默认选择
- 从发现列表构建配置

## 功能门控

- `hardware`: 启用硬件支持
- `probe`: 启用 probe-rs 集成

## 平台支持

- Linux
- macOS
- Windows

## 相关模块

- `src/peripherals/`: 外设接口

- `src/rag/`: 数据表 RAG
