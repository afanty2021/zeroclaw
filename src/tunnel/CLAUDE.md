# Tunnel Module — 隧道抽象层

**模块路径**: `src/tunnel/`

## 概述

Tunnel 模块提供隧道抽象层，支持多种隧道提供商的统一接口。

## 核心功能

### 1. Tunnel Trait

隧道接口：
```rust
#[async_trait]
pub trait Tunnel: Send + Sync {
    fn name(&self) -> &str;
    async fn start(&self, local_host: &str, local_port: u16) -> Result<String>;
    async fn stop(&self) -> Result<()>;
    async fn health_check(&self) -> bool;
    fn public_url(&self) -> Option<String>;
}
```

### 2. 支持的提供商

#### NoneTunnel
- 无隧道
- 返回本地地址

#### CloudflareTunnel
- 使用 `cloudflared` 二进制
- 需要令牌

#### TailscaleTunnel
- 使用 `tailscale` CLI
- Funnel 模式可选

#### NgrokTunnel
- 使用 `ngrok` 二进制
- 支持自定义域名

#### CustomTunnel
- 自定义命令
- 可配置健康检查

### 3. 配置

```toml
[tunnel]
provider = "cloudflare"  # | "tailscale" | "ngrok" | "custom" | "none"

[tunnel.cloudflare]
token = "your-token"

[tunnel.tailscale]
funnel = false
hostname = "myhost"

[tunnel.ngrok]
auth_token = "your-token"
domain = "my.ngrok.io"

[tunnel.custom]
start_command = "./my-tunnel start"
health_url = "http://localhost:4040/health"
url_pattern = "Public URL: (.+)"
```

### 4. 工厂函数

```rust
pub fn create_tunnel(config: &TunnelConfig) -> Result<Option<Box<dyn Tunnel>>>
```

返回 `None` 对于 `provider = "none"`。

### 5. 进程管理

共享进程包装：
```rust
pub struct TunnelProcess {
    pub child: tokio::process::Child,
    pub public_url: String,
}
```

支持优雅关闭和健康检查。

- 临时公网访问
