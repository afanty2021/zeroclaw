# Cron Module — 定时任务调度

**模块路径**: `src/cron/`

## 概述

Cron 模块实现定时任务调度系统，支持 cron 表达式、一次性任务、间隔任务和多种交付配置。

## 核心功能

### 1. 调度类型

```rust
pub enum Schedule {
    Cron {
        expr: String,       // 标准 cron 表达式
        tz: Option<String>, // 时区（如 "America/Los_Angeles"）
    },
    At {
        at: DateTime<Utc>, // 一次性执行时间
    },
    Every {
        every_ms: u64,      // 间隔毫秒数
    },
}
```

### 2. 任务类型

```rust
pub enum JobType {
    Shell(String),           // Shell 命令
    Agent {
        prompt: String,
        session_target: SessionTarget,
    },
}
```

### 3. 交付配置

```rust
pub enum DeliveryConfig {
    Silent,                  // 静默执行
    Notification(String),    // 发送到频道
}
```

### 4. CLI 命令

- `cron list`: 列出所有任务
- `cron add <expr> <cmd>`: 添加 cron 任务
- `cron add-at <timestamp> <cmd>`: 添加一次性任务
- `cron add-every <ms> <cmd>`: 添加间隔任务
- `cron once <delay> <cmd>`: 延迟后执行一次
- `cron update <id>`: 更新任务
- `cron remove <id>`: 删除任务
- `cron pause <id>`: 暂停任务
- `cron resume <id>`: 恢复任务

## 延迟解析

支持以下格式：
- `30s`: 30 秒
- `5m`: 5 分钟
- `2h`: 2 小时
- `1d`: 1 天

## 存储位置

任务存储在配置目录的 `cron.json`。

## 安全集成

更新任务时验证命令是否被安全策略允许。
