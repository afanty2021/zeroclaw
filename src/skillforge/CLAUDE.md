# SkillForge Module — 技能自动发现与集成

**模块路径**: `src/skillforge/`

## 概述

SkillForge 模块实现技能的自动发现、评估和集成管道，从外部源（如 GitHub）发现技能并自动生成兼容 ZeroClaw 的清单。

## 核心功能

### 1. SkillForgeConfig

配置：
```rust
pub struct SkillForgeConfig {
    pub enabled: bool,
    pub auto_integrate: bool,
    pub sources: Vec<String>,           // ["github", "clawhub"]
    pub scan_interval_hours: u64,       // 扫描间隔
    pub min_score: f64,                 // 最低评分
    pub github_token: Option<String>,   // GitHub PAT
    pub output_dir: String,             // 输出目录
}
```

### 2. 管道流程

```
Scout → Evaluate → Integrate
```

1. **Scout (侦察)**: 从外部源发现候选技能
2. **Evaluate (评估)**: 评分和推荐候选
3. **Integrate (集成)**: 生成兼容清单

### 3. ForgeReport

管道运行报告：
```rust
pub struct ForgeReport {
    pub discovered: usize,       // 发现数量
    pub evaluated: usize,        // 评估数量
    pub auto_integrated: usize,  // 自动集成
    pub manual_review: usize,    // 需人工审核
    pub skipped: usize,          // 跳过
    pub results: Vec<EvalResult>,
}
```

### 4. 数据源

- **GitHub**: 通过 API 搜索仓库
- **ClawHub**: 计划中
- **HuggingFace**: 计划中

### 5. 评分系统

基于以下因素：
- 代码质量
- 文档完整性
- 测试覆盖
- 安全审计

### 6. 推荐类型

- **Auto**: 自动集成
- **Manual**: 需要人工审核
- **Skip**: 跳过
