# Hooks Module — 钩子系统

**模块路径**: `src/hooks/`

## 概述

Hooks 模块实现生命周期钩子系统，在关键事件前后执行自定义逻辑。

## 核心功能

### 1. HookRunner

钩子运行器，执行：
- 预任务钩子
- 后任务钩子
- 预编辑钩子
- 后编辑钩子

### 2. HookHandler

钩子处理器接口：
```rust
pub trait HookHandler {
    async fn handle(&self, context: HookContext) -> HookResult;
}
```

### 3. HookResult

钩子执行结果：
- 成功/失败状态
- 输出数据
- 错误信息

### 4. 内置钩子

位于 `hooks/builtin/`：
- 通知钩子
- 日志钩子
- 成本追踪钩子

## 钩子类型

### 任务钩子
- `pre_task`: 任务执行前
- `post_task`: 任务执行后

### 编辑钩子
- `pre_edit`: 文件编辑前
- `post_edit`: 文件编辑后

### 会话钩子
- `session_start`: 会话开始
- `session_end`: 会话结束

## 扩展点

添加自定义钩子：
1. 实现 `HookHandler` trait
2. 注册到 `HookRunner`
3. 配置触发条件

## 相关模块

- `src/agent/`: 编排循环集成
- `src/tools/`: 工具执行钩子
