# Scripts Directory — 安装与部署脚本

**目录路径**: `scripts/`

## 概述

Scripts 目录包含 ZeroClaw 的安装、部署和 CI/CD 脚本。

## 子目录

### `scripts/ci/`
持续集成脚本，详见 [scripts/ci/CLAUDE.md](ci/CLAUDE.md)

### `scripts/release/`
发布相关脚本，详见 [scripts/release/CLAUDE.md](release/CLAUDE.md)

## 核心脚本

### 1. bootstrap.sh
一键引导脚本，执行：
- 依赖检查
- 环境配置
- 初始化设置

### 2. install.sh
安装脚本，处理：
- 二进制安装
- 配置文件创建
- 服务注册

### 3. contrib 脚本

社区贡献的辅助工具：
- `recompute_contributor_tiers.sh`: 贡献者层级计算
- 开发者统计

## 使用方式

```bash
# 一键安装
curl -sSL https://get.zeroclaw.dev | sh

# 或从源码引导
./scripts/bootstrap.sh
```

## 相关模块

- `src/service/`: 服务管理
- `src/config/`: 配置初始化
- `src/doctor/`: 系统检查
