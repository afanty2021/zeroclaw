# Scripts Directory — 安装与部署脚本

**目录路径**: `scripts/`

## 概述

Scripts 目录包含 ZeroClaw 的安装、部署和 CI/CD 脚本。

## 核心脚本

### 1. bootstrap.sh
一键引导脚本，执行：
- 依赖检查
- 环境配置
- 初始化设置

### 2. install.sh
安装脚本，处理：
- 二进制安装
- 配置文件创建
- 服务注册

### 3. CI 目录

包含持续集成脚本：
- 构建脚本
- 测试脚本
- 部署脚本

### 4. Release 目录

发布相关脚本：
- 变更日志生成
- 发布清单创建
- 多平台打包

### 5. contrib 脚本

社区贡献的辅助工具：
- 贡献者层级计算
- 开发者统计

## 使用方式

```bash
# 一键安装
curl -sSL https://get.zeroclaw.dev | sh

# 或从源码引导
./scripts/bootstrap.sh
```

## 相关模块

- `src/service/`: 服务管理
- `src/config/`: 配置初始化
- `src/doctor/`: 系统检查
